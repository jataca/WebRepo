== Description ==
Alter the font size on your sites with very large texts or make it smaller. 

Screenshots on official web page : http://newplugins.us/wp-font-resizer-widget/


This plugin can be used to increase the accessibility of sites, helping people who have visual problems to see content better. It makes use of JQuery plugin by Fred Vanelli.
